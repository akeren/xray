# X-Ray Film Digitalization System

## Features
	- Radiography
	- Radiotherapist 
	- In-messaging 
	- Merging of films
	- Analytics 

### Tools
	- PHP Procedural Programming Paradigm
	- Mysqli
	- JavaScript and
	- CSS
